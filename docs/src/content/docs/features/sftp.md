---
title: SFTP File Browser
description: Browse and manage files on your servers via SFTP
---

The built-in SFTP client lets you browse, upload, download, and manage files on your servers.

## Features

### Remote File Browser

- Navigate remote directories
- File and folder icons for easy identification
- File size and modification date display
- Hidden file support

### Local File Browser

- Access your device's local files
- Easy file selection for upload
- Download destination selection
- Recent locations history

### File Operations

- **Upload**: Transfer files from device to server
- **Download**: Transfer files from server to device
- **Rename**: Change file and folder names
- **Delete**: Remove files and folders
- **Move**: Organize files with drag-and-drop

### Connection Features

- **Auto-reconnect**: Maintain SFTP connection
- **Concurrent Transfers**: Multiple file operations
- **Transfer Queue**: Monitor upload/download progress
- **Resume Support**: Continue interrupted transfers

## Using SFTP

1. Open a server
2. Tap the **File** button
3. Browse remote files in the top panel
4. Browse local files in the bottom panel
5. Use action buttons to transfer files

## Supported Protocols

- SFTP (SSH File Transfer Protocol)
- Compatible with OpenSSH and compatible servers

## Tips

- Use **landscape mode** on mobile for split view
- Pin frequently accessed folders for quick access
- Check file permissions before attempting transfers
