The SSH Terminal provides complete command-line access to your servers with a powerful, feature-rich interface.

## Features

### Multi-Tab Support

- Open multiple SSH sessions in tabs
- Switch between servers quickly
- Each tab maintains its own session state
- Tabs persist between app launches

### Virtual Keyboard (Mobile)

On mobile devices, a specialized virtual keyboard provides quick access to:

- Modifier keys (Ctrl, Alt, Shift, Esc)
- Function keys (F1-F12)
- Arrow keys
- Tab, Home, End, Insert, Delete
- Common terminal shortcuts

### Customizable Appearance

- Background color and opacity
- Font size adjustment
- Text color options
- Blur effects for modern look

## Terminal Actions

- **Copy/Paste**: Easy text selection and clipboard integration
- **Clear Screen**: Reset the terminal view
- **Save Session**: Preserve terminal state
- **Export Logs**: Save terminal output

## Connection Management

- **Auto-reconnect**: Automatically reconnect on connection loss
- **Keep-alive**: Maintain connection during inactivity
- **Connection Status**: Visual indicator of connection health

## Keyboard Shortcuts (Desktop)

- **Ctrl+T**: New tab
- **Ctrl+W**: Close tab
- **Ctrl+Tab**: Next tab
- **Ctrl+Shift+Tab**: Previous tab
- **Ctrl+C**: Copy (when text selected)
- **Ctrl+V**: Paste

## Tips

- Use **snippets** for frequently used commands
- Enable **biometric authentication** for quick terminal access
- Customize **font size** for better readability on different devices
