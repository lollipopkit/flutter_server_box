Flutter Server Box proporciona un monitoreo integral en tiempo real de la salud y el rendimiento de su servidor.

## Tarjetas de Estado

La página de detalles del servidor muestra tarjetas de estado configurables para diferentes métricas del sistema. Puede habilitar/deshabilitar tarjetas en la configuración.

### Monitoreo de CPU

- Porcentaje de uso de CPU en tiempo real
- Desglose de uso de CPU por núcleo
- Gráficos de uso histórico
- Información de frecuencia de CPU

### Monitoreo de Memoria

- **Uso de RAM**: Memoria usada vs total con porcentaje
- **Uso de Swap**: Utilización de memoria de intercambio
- Indicadores de presión de memoria
- Tendencias históricas de memoria

### Monitoreo de Disco

- Uso de puntos de montaje con porcentaje
- Espacio total, usado y libre
- Estadísticas de E/S
- Soporte para múltiples discos

### Monitoreo de Red

- Velocidades de carga/descarga en tiempo real
- Gráficos de uso de ancho de banda
- Estadísticas de interfaz de red
- Datos totales transferidos

### Métricas Avanzadas

- **Estado de GPU**: Monitoreo de GPU NVIDIA y AMD
- **Temperatura**: Temperaturas de CPU, GPU y sistema
- **Sensores**: Velocidades de ventiladores, voltajes y otros datos de sensores
- **S.M.A.R.T**: Monitoreo de salud del disco
- **Batería**: Estado de UPS o batería (si está disponible)

## Personalización de la Pantalla

### Reordenar Tarjetas

1. Vaya a Configuración
2. Seleccione Configuración del Servidor
3. Arrastre las tarjetas para reordenarlas en la página de detalles del servidor

### Habilitar/Deshabilitar Tarjetas

1. Abra la página de detalles de un servidor
2. Toque el botón de editar/menú
3. Active o desactive las tarjetas individuales

## Actualización Automática

- Las tarjetas de estado se actualizan automáticamente
- El intervalo de actualización es configurable en los ajustes
- Actualización manual disponible con el gesto de deslizar para refrescar

## Gráficos y Visualizaciones

- **Gráficos de Líneas**: Tendencias de datos históricos
- **Gráficos de Indicador**: Porcentaje de uso actual
- **Codificación de Colores**: Indicadores visuales para niveles de estado
- **Zoom**: Pellizcar para hacer zoom en los gráficos para vistas detalladas
