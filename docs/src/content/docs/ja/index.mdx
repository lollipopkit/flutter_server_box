## 特徴

<CardGrid stagger>
	<Card title="リアルタイム監視" icon="chart">
		CPU、メモリ、ディスク、ネットワーク、GPU、温度を美しいリアルタイムチャートで監視します。
	</Card>
	<Card title="SSH ターミナル" icon="terminal">
		マルチタブ対応、モバイルデバイス用仮想キーボードを備えたフル機能の SSH ターミナル。
	</Card>
	<Card title="SFTP ファイルブラウザ" icon="folder">
		内蔵の SFTP クライアントとローカルファイルブラウザでサーバー上のファイルを管理。
	</Card>
	<Card title="Docker 管理" icon="box">
		直感的なインターフェースで Docker コンテナの起動、停止、監視が可能。
	</Card>
	<Card title="クロスプラットフォーム" icon="device-mobile">
		iOS、Android、macOS、Linux、Windows、watchOS で利用可能。
	</Card>
	<Card title="12以上の言語" icon="globe">
		英語、中国語、ドイツ語、フランス語など、フルローカライズをサポート。
	</Card>
</CardGrid>
