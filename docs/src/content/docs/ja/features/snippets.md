---
title: スニペット
description: カスタムシェルコマンドの保存と実行
---

スニペットを使用すると、頻繁に使用するシェルコマンドを保存して素早く実行できます。

## スニペットの作成

1. **[スニペット]** タブに移動します
2. **+** ボタンをタップします
3. スニペットの詳細を入力します：
   - **名前**: スニペットの分かりやすい名前
   - **コマンド**: 実行するシェルコマンド
   - **説明**: オプションのメモ
4. スニペットを保存します

## スニペットの使用

1. サーバーを開きます
2. **[スニペット]** ボタンをタップします
3. 実行するスニペットを選択します
4. ターミナルで出力を確認します

## スニペットの機能

- **クイック実行**: ワンタップでコマンドを実行
- **変数**: サーバー固有の変数を使用
- **整理**: 関連するスニペットをグループ化
- **インポート/エクスポート**: デバイス間でスニペットを共有
- **同期**: オプションのクラウド同期

## スニペットの例

### システムアップデート
```bash
sudo apt update && sudo apt upgrade -y
```

### ディスククリーンアップ
### Disk Cleanup
```bash
sudo apt autoremove -y && sudo apt clean
```

### Docker クリーンアップ
```bash
docker system prune -a
```

### システムログの表示
```bash
journalctl -n 50 -f
```

## ヒント

- 簡単に識別できるように **分かりやすい名前** を使用してください
- 複雑なコマンドには **コメント** を追加してください
- スニペットとして保存する前にコマンドをテストしてください
- カテゴリやサーバーの種類ごとにスニペットを整理してください
