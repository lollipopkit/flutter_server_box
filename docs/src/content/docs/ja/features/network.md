Server Box には、テストと診断のためのいくつかのネットワークツールが含まれています。

## iPerf

デバイスとサーバーの間でネットワーク速度テストを実行します。

### 機能

- **アップロード/ダウンロード速度**: 帯域幅のテスト
- **サーバーモード**: サーバーを iPerf サーバーとして使用
- **クライアントモード**: iPerf サーバーに接続
- **カスタムパラメータ**: 期間、パラレルストリームなど

### 使用方法

1. サーバーを開く
2. **iPerf** をタップ
3. サーバーまたはクライアントモードを選択
4. パラメータを設定
5. テストを開始

## Ping

ネットワークの接続性と遅延をテストします。

### 機能

- **ICMP Ping**: 標準的な Ping ツール
- **パケット数**: パケット数を指定
- **パケットサイズ**: カスタムパケットサイズ
- **インターバル**: Ping 間の時間

### 使用方法

1. サーバーを開く
2. **Ping** をタップ
3. ターゲットホストを入力
4. パラメータを設定
5. Ping を開始

## Wake on LAN

マジックパケットを介してリモートサーバーを起動します。

### 機能

- **MAC アドレス**: ターゲットデバイスの MAC
- **ブロードキャスト**: ブロードキャストマジックパケットを送信
- **保存されたプロファイル**: WoL 設定を保存

### 要件

- ターゲットデバイスが Wake-on-LAN をサポートしている必要があります
- BIOS/UEFI で WoL が有効になっている必要があります
- デバイスがスリープまたはソフトオフ状態である必要があります
- デバイスが同じネットワーク上にあるか、ブロードキャスト経由で到達可能である必要があります

## ヒント

- iPerf を使用してネットワークのボトルネックを診断します
- 複数のホストに Ping を送信して遅延を比較します
- 頻繁に起動するデバイスの WoL プロファイルを保存します
