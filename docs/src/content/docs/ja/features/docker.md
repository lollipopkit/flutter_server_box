---
title: Docker 管理
description: Docker コンテナの監視と管理
---

Server Box は、サーバー上の Docker コンテナを管理するための直感的なインターフェースを提供します。

## 機能

### コンテナリスト

- すべてのコンテナを表示（実行中および停止中）
- コンテナ ID と名前の表示
- イメージ情報
- ステータスインジケーター
- 作成時間

### コンテナアクション

- **開始**: 停止中のコンテナを起動
- **停止**: 実行中のコンテナを正常に停止
- **再起動**: コンテナを再起動
- **削除**: コンテナを削除
- **ログを表示**: コンテナログを確認
- **インスペクト**: コンテナの詳細を表示

### コンテナの詳細

- 環境変数
- ポートマッピング
- ボリュームマウント
- ネットワーク設定
- リソース使用状況

## 要件

- サーバーに Docker がインストールされている必要があります
- SSH ユーザーに Docker 権限が必要です
- 非 root ユーザーの場合は、docker グループに追加してください：
  ```bash
  sudo usermod -aG docker your_username
  ```

## クイックアクション

- シングルタップ: コンテナの詳細を表示
- 長押し: クイックアクションメニュー
- スワイプ: クイック開始/停止
- 一括選択: 複数のコンテナ操作

## ヒント

- **自動更新**を使用して、コンテナのステータス変更を監視します
- 実行中/停止中のコンテナでフィルタリング
- 名前または ID でコンテナを検索
