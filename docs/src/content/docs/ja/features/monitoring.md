title: サーバー監視
description: 美しいチャートによるリアルタイムのサーバー状態監視
---

Server Box は、サーバーの健全性とパフォーマンスに関する包括的なリアルタイム監視を提供します。

## ステータスカード

サーバーの詳細ページには、さまざまなシステムメトリクスの設定可能なステータスカードが表示されます。設定でカードの有効化/無効化を切り替えることができます。

### CPU 監視

- リアルタイムの CPU 使用率
- コアごとの CPU 使用率の詳細
- 履歴使用状況チャート
- CPU 周波数情報

### メモリ監視

- **RAM 使用量**: 使用済み vs 合計メモリ（パーセンテージ付き）
- **スワップ使用量**: スワップメモリの使用状況
- メモリプレッシャーインジケーター
- 履歴メモリトレンド

### ディスク監視

- マウントポイントごとの使用率
- 合計、使用済み、空き容量
- I/O 統計
- 複数ディスクのサポート

### ネットワーク監視

- リアルタイムのアップロード/ダウンロード速度
- 帯域幅使用状況チャート
- ネットワークインターフェース統計
- 総データ転送量

### 高度なメトリクス

- **GPU ステータス**: NVIDIA および AMD GPU の監視
- **温度**: CPU、GPU およびシステムの温度
- **センサー**: ファンスピード、電圧、その他のセンサーデータ
- **S.M.A.R.T**: ディスクの健全性監視
- **バッテリー**: UPS またはバッテリーの状態（利用可能な場合）

## 表示のカスタマイズ

### カードの並べ替え

1. [設定] に移動します
2. [サーバー設定] を選択します
3. カードをドラッグして、サーバー詳細ページでの順序を変更します

### カードの有効化/無効化

1. サーバーの詳細ページを開きます
2. 編集/メニューボタンをタップします
3. 個々のカードのオン/オフを切り替えます

## 自動更新

- ステータスカードは自動的に更新されます
- 更新間隔は設定で変更可能です
- プルダウン操作による手動更新も可能です

## チャートと視覚化

- **折れ線チャート**: 履歴データのトレンド
- **ゲージチャート**: 現在の使用率
- **カラーコーディング**: ステータスレベルの視覚的なインジケーター
- **ズーム**: チャートをピンチしてズームし、詳細を表示できます
