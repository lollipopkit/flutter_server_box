---
title: 代码片段
description: 保存并执行自定义 Shell 命令
---

代码片段允许您保存常用的 Shell 命令以便快速执行。

## 创建代码片段

1. 前往 **代码片段 (Snippets)** 选项卡
2. 点击 **+** 按钮
3. 填写片段详情：
   - **名称**：代码片段的友好名称
   - **命令**：要执行的 Shell 命令
   - **描述**：可选备注
4. 保存代码片段

## 使用代码片段

1. 打开一个服务器
2. 点击 **代码片段 (Snippet)** 按钮
3. 选择要执行的代码片段
4. 在终端中查看输出

## 代码片段功能

- **快速执行**：一键执行命令
- **变量**：使用服务器特定的变量
- **组织**：将相关片段分组
- **导入/导出**：在设备之间分享代码片段
- **同步**：可选的云同步

## 示例代码片段

### 系统更新
```bash
sudo apt update && sudo apt upgrade -y
```

### 磁盘清理
```bash
sudo apt autoremove -y && sudo apt clean
```

### Docker 清理
```bash
docker system prune -a
```

### 查看系统日志
```bash
journalctl -n 50 -f
```

## 提示

- 使用**描述性名称**以便识别
- 为复杂命令添加**注释**
- 在保存为片段之前先测试命令
- 按类别或服务器类型组织代码片段
