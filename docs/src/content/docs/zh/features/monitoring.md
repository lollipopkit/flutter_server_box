Server Box 提供对服务器健康状况和性能的全面实时监控。

## 状态卡片

服务器详情页面显示了不同系统指标的可配置状态卡片。您可以在设置中启用/禁用卡片。

### CPU 监控

- 实时 CPU 使用百分比
- 每核 CPU 使用率分解
- 历史使用图表
- CPU 频率信息

### 内存 监控

- **RAM 使用情况**：已用内存 vs 总内存（带百分比）
- **Swap 使用情况**：Swap 内存利用率
- 内存压力指标
- 历史内存趋势

### 磁盘监控

- 挂载点使用百分比
- 总空间、已用空间和剩余空间
- I/O 统计
- 多磁盘支持

### 网络监控

- 实时上传/下载速度
- 带宽使用图表
- 网络接口统计
- 总数据传输量

### 高级指标

- **GPU 状态**：NVIDIA 和 AMD GPU 监控
- **温度**：CPU、GPU 和系统温度
- **传感器**：风扇速度、电压和其他传感器数据
- **S.M.A.R.T**：磁盘健康监控
- **电池**：UPS 或电池状态（如果可用）

## 自定义显示

### 重新排序卡片

1. 前往设置
2. 选择服务器设置
3. 在服务器详情页面上拖动卡片以重新排序

### 启用/禁用卡片

1. 打开服务器详情页面
2. 点击编辑/菜单按钮
3. 切换各个卡片的开关

## 自动刷新

- 状态卡片会自动刷新
- 刷新间隔可在设置中配置
- 支持通过下拉刷新手势手动刷新

## 图表和可视化

- **折线图**：历史数据趋势
- **仪表盘**：当前使用百分比
- **颜色编码**：状态级别的视觉指示
- **缩放**：在图表上捏合缩放以查看详细视图
