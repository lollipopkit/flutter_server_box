Flutter Server Box 包含多种用于测试和诊断的网络工具。

## iPerf

在您的设备和服务器之间进行网络速度测试。

### 功能特性

- **上传/下载速度**：测试带宽
- **服务器模式**：将服务器用作 iPerf 服务器
- **客户端模式**：连接到 iPerf 服务器
- **自定义参数**：持续时间、并行流等。

### 使用方法

1. 打开服务器
2. 点击 **iPerf**
3. 选择服务器或客户端模式
4. 配置参数
5. 开始测试

## Ping

测试网络连接性和延迟。

### 功能特性

- **ICMP Ping**：标准 Ping 工具
- **数据包计数**：指定数据包数量
- **数据包大小**：自定义数据包大小
- **间隔**：Ping 之间的等待时间

### 使用方法

1. 打开服务器
2. 点击 **Ping**
3. 输入目标主机
4. 配置参数
5. 开始 Ping

## 网络唤醒 (WoL)

通过幻数据包唤醒远程服务器。

### 功能特性

- **MAC 地址**：目标设备 MAC
- **广播**：发送广播幻数据包
- **保存的配置集**：存储 WoL 配置

### 要求

- 目标设备必须支持网络唤醒
- 必须在 BIOS/UEFI 中启用 WoL
- 设备必须处于睡眠/软关机状态
- 设备必须在同一网络中或可通过广播访问

## 提示

- 使用 iPerf 诊断网络瓶颈
- Ping 多个主机以比较延迟
- 为经常唤醒的设备保存 WoL 配置集
