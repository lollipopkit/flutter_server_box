title: Docker 管理

---

Flutter Server Box 为管理服务器上的 Docker 容器提供了一个直观的界面。

## 功能特性

### 容器列表

- 查看所有容器（运行中和已停止）
- 显示容器 ID 和名称
- 镜像信息
- 状态指示器
- 创建时间

### 容器操作

- **启动**：启动已停止的容器
- **停止**：优雅地停止运行中的容器
- **重启**：重启容器
- **删除**：删除容器
- **查看日志**：查看容器日志
- **检查**：查看容器详情

### 容器详情

- 环境变量
- 端口映射
- 卷挂载
- 网络配置
- 资源使用情况

## 要求

- 服务器必须安装 Docker
- SSH 用户必须具有 Docker 权限
- 对于非 root 用户，请添加到 docker 组：
  ```bash
  sudo usermod -aG docker your_username
  ```

## 快捷操作

- 单击：查看容器详情
- 长按：快捷操作菜单
- 滑动：快速启动/停止
- 批量选择：多个容器操作

## 提示

- 使用**自动刷新**来监控容器状态变化
- 按运行中/已停止容器进行过滤
- 按名称或 ID 搜索容器
