## 进程管理

查看并管理服务器上运行的进程。

### 进程列表

- 所有运行中的进程及其详细信息
- PID（进程 ID）
- CPU 和内存使用情况
- 用户所有权
- 进程命令

### 进程操作

- **结束进程 (Kill)**：终止进程
- **过滤**：按名称或用户过滤
- **排序**：按 CPU、内存或 PID 排序
- **搜索**：查找特定进程

## Systemd 服务

管理 systemd 服务以进行服务控制。

### 服务列表

- 所有 systemd 服务
- 活跃/非活跃状态
- 已启用/已禁用状态
- 服务描述

### 服务操作

- **启动**：启动已停止的服务
- **停止**：停止运行中的服务
- **重启**：重启服务
- **启用**：启用开机自启
- **禁用**：禁用开机自启
- **查看状态**：检查服务状态和日志
- **重载**：重载服务配置

## 要求

- SSH 用户必须具有适当的权限
- 对于服务管理：可能需要 `sudo` 权限
- 查看进程：通常标准用户权限即可

## 提示

- 使用进程列表识别资源占用大户
- 检查服务日志以进行故障排除
- 通过自动刷新监控关键服务
