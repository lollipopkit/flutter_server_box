Server Box 支持管理 Proxmox VE 虚拟化平台。

## 功能特性

### 虚拟机 (VM) 管理

- **列出虚拟机**：查看所有虚拟机
- **虚拟机状态**：检查运行/停止状态
- **虚拟机操作**：启动、停止、重启虚拟机
- **虚拟机详情**：查看配置和资源占用

### 容器 (LXC) 管理

- **列出容器**：查看所有 LXC 容器
- **容器状态**：监控容器状态
- **容器操作**：启动、停止、重启容器
- **控制台访问**：通过终端访问容器

### 节点监控

- **资源使用情况**：CPU、内存、磁盘、网络
- **节点状态**：检查节点健康状况
- **集群视图**：多节点集群概览

## 设置

### 添加 PVE 服务器

1. 按正常的 SSH 连接添加服务器
2. 确保用户具有 PVE 权限
3. 从服务器详情页面访问 PVE 功能

### 所需权限

PVE 用户需要：

- **VM.Audit**：查看虚拟机状态
- **VM.PowerMgmt**：启动/停止虚拟机
- **VM.Console**：控制台访问

权限设置示例：

```bash
pveum useradd myuser -password mypass
pveum aclmod /vms -user myuser@pve -role VMAdmin
```

## 使用方法

### 虚拟机管理

1. 打开带有 PVE 的服务器
2. 点击 **PVE** 按钮
3. 查看虚拟机列表
4. 点击虚拟机查看详情
5. 使用操作按钮进行管理

### 容器管理

1. 打开带有 PVE 的服务器
2. 点击 **PVE** 按钮
3. 切换到“容器”选项卡
4. 查看并管理 LXC 容器

### 监控

- 实时资源使用情况
- 历史数据图表
- 支持多节点

- 资源监控

### 计划中

## 要求

- **PVE 版本**：6.x 或 7.x
- **访问**：对 PVE 主机的 SSH 访问权限
- **权限**：适当的 PVE 用户角色
- **网络**：对 PVE API 的连接能力（通过 SSH）

## 提示

- 使用具有有限权限的**专用 PVE 用户**
- 监控**资源使用情况**以获得最佳性能
- 在维护前检查**虚拟机状态**
- 在进行重大更改前使用**快照**
